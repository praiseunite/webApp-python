# Class Task: Creating Your First Module

## Objective
Practice separating your code into multiple files by creating and importing a custom module.

## Instructions
1. Create a new file called `my_greetings.py`.
2. Inside `my_greetings.py`, define a function `say_hi(name)` that prints `"Hi [name]! Welcome to the module."`
3. Save the file.
4. Now, open your main script (e.g., `class_task_solution.py` or the Python interactive shell).
5. `import my_greetings`
6. Call the function using `my_greetings.say_hi("Alice")`.

You can view the main script solution in `class_task_solution.py` and the module in `my_greetings.py`.
