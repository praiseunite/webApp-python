# Assignment: Mini-Project 1 Kickoff (The Profile Card)

## Objective
We are going to build an interactive CLI (Command Line Interface) application over the next few sessions. For your first assignment, you will set up the base profile for a user.

## Instructions
Open the provided `assignment_starter.py` and complete the following:
1. Write a multi-line comment at the top of the file describing what the script does.
2. Create **four** properly named variables storing the following information about yourself or a fictional character:
   - First Name
   - Age
   - Favorite Color
   - Are you a programmer? (True/False)
3. Print these variables out in a formatted, easy-to-read way. 
4. Ensure your code is well-commented.
