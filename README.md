# Application Based Programming in Python 🐍

Welcome to the **Application Based Programming in Python** course repository! This curriculum is designed to take students from absolute beginners to building interactive web applications, scraping data, and training machine learning models—all through a strictly practical, project-based approach.

## 🎯 Course Objectives
By the end of this course, students will be able to:
- Gain an in-depth understanding of the basics of Python programming.
- Organize and write modular, reusable code using Functions and Packages.
- Perform Web Scraping using Python libraries.
- Build Interactive Desktop GUIs with Tkinter and Web Apps with Flask.
- Connect to and interact with MySQL Databases.
- Visualize data using Pandas and apply Regression Techniques in Machine Learning.

## 🗂️ Repository Structure
This repository is organized session-by-session. Inside each session folder, you will find:
- 📖 **`Lecture_Notes.md`**: Theoretical explanations, code-alongs, diagrams, and YouTube video recommendations.
- 🏋️ **`Class_Task.md` & `class_task_solution.py`**: Small in-class exercises to reinforce immediate learning, along with their solutions.
- 🚀 **`Assignment.md` & `assignment_starter.py`**: Take-home homework that builds toward larger, overarching mini-projects.

## 🗺️ Course Syllabus

### Phase 1: Core Python & Logic
- **Session 1:** Basics of Python (Syntax, Variables, Identifiers)
- **Session 2:** Programming Language Construct (Data Types, Operators, Control Flow)
- **Session 3:** Introduction to Functions (Scope, Parameters, Returns)
- **Session 4:** Functions, Modules, and Packages (Recursion, Lambdas, Imports)

### Phase 2: Interacting with the World *(Upcoming)*
- **Session 5:** File Handling and Exception Handling
- **Session 6:** Regular Expressions and Tokenization
- **Session 7:** Web Development Using Flask
- **Session 8 & 9:** Web Scraping & GUI Programming in Tkinter

### Phase 3 & 4: Data Science & Machine Learning *(Upcoming)*
- **Session 10 & 11:** Database Integration (MySQL) & Data Handling (Pandas)
- **Session 12 & 13:** Data Visualization & ML Processing Techniques
- **Session 14 & 15:** Machine Learning Types & Regression Techniques

## 🚀 How to Use This Course

### Prerequisites
- Install [Python 3.x](https://www.python.org/downloads/) on your machine.
- Install a code editor like [Visual Studio Code](https://code.visualstudio.com/) or PyCharm.

### Getting Started
1. **Clone the repository:**
   ```bash
   git clone https://github.com/praiseunite/webApp-python.git
   cd webApp-python
   ```
2. **Navigate to the current session folder:**
   ```bash
   cd Session_01_Basics
   ```
3. Read the `Lecture_Notes.md` (tip: view it on GitHub or use a Markdown previewer in VS Code).
4. Complete the `Class_Task` in your editor, then open `assignment_starter.py` to begin your homework project!

Happy Coding! 🎉
