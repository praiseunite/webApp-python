# Class Task: My First Plots 📈

## Objective
Your task is to write a script that generates two simple plots using `matplotlib.pyplot`: a Line Plot and a Bar Chart.

## Instructions
1. Open `class_task_starter.py`.
2. The data for months, revenue, and expenses has been provided for you.
3. **Task 1: Line Plot**
   - Plot the `revenue` over the `months`.
   - Add a title: "Monthly Revenue".
   - Add x and y labels.
   - Show the plot.
4. **Task 2: Bar Chart**
   - Plot the `expenses` over the `months` as a bar chart.
   - Add a title: "Monthly Expenses".
   - Add x and y labels.
   - Give the bars a specific color (e.g., 'orange' or 'red').
   - Show the plot.

## Expected Output
When you run the script, two separate windows should pop up sequentially (or at the same time if using subplots, but sequential is fine for this task). The first will be a line plot showing a trend, and the second will be a bar chart showing discrete values.
