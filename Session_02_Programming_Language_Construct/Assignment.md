# Assignment: Mini-Project 1 Expansion (Interactive Profile)

## Objective
Make the profile application from Session 1 interactive by taking user input and using loops/conditions.

## Instructions
Open `assignment_starter.py` and complete the following:
1. Use the `input("Prompt: ")` function to ask the user for their Name and Age.
   - *Hint: Remember to convert the age string into an integer using `int()`!*
2. Write a `for` loop using `range(3)` that asks the user: `"Name one of your favorite hobbies:"` and prints it back to them.
3. Add a control flow statement at the end: 
   - If their age is `>= 18`, print `"Profile created for an adult."` 
   - Else, print `"Profile created for a minor."`
