# Class Task: The Modular Calculator

## Objective
Practice creating and calling functions with parameters and return values.

## Instructions
Open your Python environment and write a script that does the following:
1. Create a function called `multiply_numbers` that takes two parameters (`a` and `b`).
2. Inside the function, multiply `a` and `b` and `return` the result.
3. Create another function called `divide_numbers` that takes two parameters (`x` and `y`).
4. Inside the function, divide `x` by `y` and `return` the result.
5. Outside the functions, call `multiply_numbers(10, 5)` and save the result in a variable called `product`.
6. Call `divide_numbers(20, 4)` and save the result in a variable called `quotient`.
7. Print both results to the screen.

You can view the solution in `class_task_solution.py`.
